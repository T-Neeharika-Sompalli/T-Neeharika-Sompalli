## Featured Projects

<details>
<summary><b>💬 Exploring Mental Health Narratives on Reddit — NLP at Scale</b></summary>

<br/>

An end-to-end NLP data engineering and topic discovery system that processes over 10,000 Reddit posts and comments from mental health communities (anxiety, ADHD, depression, stress). The pipeline spans automated ingestion through semantic topic modeling, designed to surface latent thematic patterns across large-scale unstructured social text corpora.

<div align="center">

| Attribute | Details |
|:---|:---|
| **Stack** | Python, SQL, Sentence-BERT, BERTopic, UMAP, HDBSCAN, spaCy, NLTK |
| **Scale** | 10,000+ Reddit posts across multiple mental health community subreddits |
| **Performance** | ~35% reduction in semantic overlap across topics; 70% reduction in manual preprocessing effort |
| **Security** | Anonymized community data; no PII retained in processed outputs |
| **Impact** | Enables large-scale thematic discovery in mental health discourse for research and policy |
| **Repository** | [View on GitHub](https://github.com/T-Neeharika-Sompalli) |

</div>

Automated ingestion and preprocessing pipelines handle Named Entity Recognition, document chunking, and embedding generation via Sentence-BERT. BERTopic with UMAP dimensionality reduction and HDBSCAN clustering surfaces coherent topic clusters. Evaluation uses coherence scores and topic diversity metrics. Analytical data models store embeddings and topic metadata for reproducible downstream querying — a production-grade architecture for social text intelligence.

</details>

<details>
<summary><b>🧠 Hybrid Face Recognition Using Deep and Traditional Features</b></summary>

<br/>

A hybrid biometric recognition system that fuses CNN-based deep embeddings with classical handcrafted feature descriptors (LBP, HOG) to achieve superior recognition accuracy over standalone deep or classical approaches. Designed for constrained settings where pose variation and lighting inconsistency challenge pure deep learning pipelines.

<div align="center">

| Attribute | Details |
|:---|:---|
| **Stack** | Python, TensorFlow/Keras, OpenCV, Scikit-learn, NumPy |
| **Scale** | Multi-condition evaluation across standard face recognition benchmarks |
| **Performance** | Hybrid feature fusion outperforms standalone CNN and classical descriptor baselines |
| **Security** | Biometric-grade identity verification pipeline with modular classification head |
| **Impact** | Applicable to security, surveillance, and low-data identity systems |
| **Repository** | [View on GitHub](https://github.com/T-Neeharika-Sompalli/Hybrid-Face-Recognition-Using-Deep-and-Traditional-Features) |

</div>

Deep feature maps from a fine-tuned convolutional backbone are concatenated with spatial texture descriptors before classification. Image preprocessing, face detection, feature fusion, and classification modules are fully decoupled for modularity and reuse. Performance evaluation compares hybrid, deep-only, and classical-only pipelines with documented accuracy improvements.

</details>
